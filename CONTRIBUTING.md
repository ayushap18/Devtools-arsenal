# Contributing to Devtools Arsenal

Thank you for your interest in contributing! This project thrives on community contributions. Whether you want to add a new tool, fix a broken link, or improve the documentation, your help is welcome.

## How to Contribute

### Adding a New Tool

1. **Fork** this repository
2. **Find the right category** in `README.md` for your tool
3. **Add your entry** following this format:

```markdown
- [Tool Name](https://github.com/owner/repo) - Brief one-line description of what the tool does. [![GitHub stars](https://img.shields.io/github/stars/owner/repo?style=flat-square)](https://github.com/owner/repo/stargazers) [![Last commit](https://img.shields.io/github/last-commit/owner/repo?style=flat-square)](https://github.com/owner/repo/commits)
```

4. **Submit a Pull Request** with a clear description

### Requirements for Tools

- Must be **open-source** with a public repository
- Must be **actively maintained** (commit within the last 12 months)
- Must be a **developer productivity tool** (editors, CLI tools, DevOps, etc.)
- Must have a **clear description** of what it does
- Should have at least **50+ stars** on GitHub (exceptions for exceptional tools)

### Fixing Issues

- **Broken links**: Submit a PR with the corrected URL
- **Outdated descriptions**: Update with current information
- **Miscategorized tools**: Move to the appropriate category
- **Duplicate entries**: Remove the duplicate, keep the more complete entry

### Style Guide

- Keep descriptions concise (one line, under 120 characters)
- Use sentence case for descriptions
- Don't use promotional language ("the best", "amazing", "awesome")
- Include platform tags where applicable
- Alphabetical order within each category

### Creating Issues

- Use the issue tracker for suggestions, broken links, or category proposals
- Tag issues appropriately: `add-tool`, `broken-link`, `category`, `enhancement`

## Code of Conduct

Be respectful, constructive, and inclusive. We're all here to build a useful resource for the developer community.
